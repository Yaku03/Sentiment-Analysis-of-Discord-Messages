About this Application



#missed-connections is a channel in Melee Online, a popular discord server for the game Super Smash Brothers Melee.

This server aims to connect people through online matchmaking, and the #missed-connections channel allows people to reach out to others that they have just played, as the game itself provides very limited chat functionality.

However the messages in this channel tend to get a bit, uh...

freaky.



Using a sentiment analysis model made with Python and TensorFlow, this website displays the number of positive, neutral, and negative messages that were said in the channel. The model analyzes the messages in real time.

The website uses the Flask Python framework for the backend, and a simple .csv file as a databse to pull the sentiment counts from the model.
